# ASAUViA-style Home + Loader (React)

This is a lightweight Vite + React template that includes:
- Full-screen animated Loader
- Futuristic techfest-style Home screen (hero, sections, cards, timeline)
- Pure CSS (no Tailwind) so you can drop it into any stack

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Customize
- Replace the logo: `src/components/LogoMark.jsx`
- Edit sections: `src/pages/Home.jsx`
- Tweak theme colors: `src/styles/global.css`
